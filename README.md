# GitPeek 👀

GitPeek is a lightweight, static web app that lets you quickly **peek inside any GitHub repository** without cloning or downloading it.

Paste a repo link → instantly browse files → preview content → download what you need.

---

## 🚀 Features

- 🔗 Paste any GitHub repo URL
- 🌲 Collapsible file tree (like VS Code / GitHub)
- 👁️ Live file preview:
  - Images (.png, .jpg, .gif, .webp)
  - Markdown (.md) – beautifully rendered
  - HTML – sandboxed preview
  - Code files (.js, .css, .json, .txt) with syntax highlighting
- ⬇️ Download single files instantly
- 📦 Download full repo as ZIP
- 🌙 Dark / ☀️ Light mode
- ⚡ Fast, static, no backend required

---

## 🧠 How It Works

GitPeek uses public GitHub URLs to:

- Fetch repository structure
- Render files directly in the browser
- Use GitHub’s archive links for downloads

No cloning. No backend. No API keys.
